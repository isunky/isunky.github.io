---
title: 重建个人站点的几个取舍
description: 为什么首版选择 Astro、Markdown 和 GitHub Pages，而不是更重的内容系统。
date: 2026-06-11
tags:
  - 方法
  - 网站
---

个人站点的首版不需要后台系统，也不需要复杂的发布流程。内容能写、能检索、能归档、能通过 GitHub Pages 稳定发布，就足够了。

Astro 适合这个阶段：页面是静态的，Markdown 可以直接作为内容源，后续也能逐步增加搜索、RSS、主题切换和产品页。

更重要的是，首版要让内容结构先稳定下来。视觉可以继续调整，但日志和产品的边界要先清楚。
